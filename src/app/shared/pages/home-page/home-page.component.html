Bienvenido a EchoVault

EchoVault es una aplicación que te permite conocer a tus artistas favoritos, o tus canciones más escuchadas.
Utilizamos la API de Spotify, lo que quiere decir que la aplicación es completamente segura.
Se requiere de una suscripción a Spotify Premium para poder utilizar la aplicación.

La aplicacion utiliza la API de Spotify para obtener informacion de tus artistas favoritos, puede que no funcione correctamente si no tienes una cuenta de Spotify Premium.
